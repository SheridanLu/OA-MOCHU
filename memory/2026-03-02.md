# 2026-03-02 工作日志

## 创建 OA Builder Framework

根据用户要求，基于 Anthropic "Effective harnesses for long-running agents" 论文创建了一套用于构建小型办公 OA 系统的 Agent 框架。

### 完成内容

1. **SKILL.md** - 技能说明和核心理念
2. **prompts/initializer.md** - Initializer Agent 提示词
3. **prompts/coder.md** - Coding Agent 提示词
4. **templates/feature-list.json** - 26 个功能点清单
5. **templates/progress.md** - 进度日志模板
6. **templates/init.sh** - 开发环境启动脚本
7. **templates/database/init.sql** - 数据库 Schema
8. **README.md** - 使用说明

### 功能模块

- 用户认证（登录/退出/记住我）
- 仪表盘（待办/日程/通知）
- 请假管理（申请/审批/余额）
- 报销管理（提交/审批/打款）
- 公告通知（发布/阅读/已读）
- 通讯录（搜索/部门筛选）
- 权限管理（角色/部门/账号）

### 核心设计原则

1. 双 Agent 模式（Initializer + Coder）
2. Feature List 驱动开发
3. 每个 session 只做一个功能
4. 端到端测试验证
5. 保持干净状态
