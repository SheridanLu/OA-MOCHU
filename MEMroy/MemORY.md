## 技术栈
- **前端**: Vite + React 18 + Ant Design 5
- **后端**: Express + SQLite (better-sqlite3)
- **认证**: JWT
- **部署**: systemd + PM2

## 数据库设计
- 25个表
- 完整RBAC权限系统
- 多级审批流引擎

- 定时任务（资质预警/对账单生成）

## 核心业务规则
- **项目编号**: 实体P+年份+6位序号（10位）， 虚拟v+年份+4位序号（8位）
- **合同编号**: HT+年月+4位序号（8位）
- **合同时间窗**: 每月25-30日
- **零散采购预警**: 超过批量采购1.5%
- **底价系统**: 追踪物资历史最低价
- **中止规则**: 虚拟项目中止必须选择成本下挂目标
