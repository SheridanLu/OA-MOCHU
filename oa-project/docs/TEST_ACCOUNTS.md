## 📝 测试建议

### 1. 测试权限控制
- 用不同账号登录，测试权限是否正确
- 尝试执行无权限的操作，验证是否被拦截

### 2. 测试业务流程
- **采购流程**: 采购员创建项目 → 预算员审批 → 总经理审批
- **合同流程**: 采购员创建合同 → 财务审批 → 总经理审批
- **报销流程**: 普通员工申请 → 财务审批

### 3. 测试数据隔离
- 不同角色看到的数据范围
- 部门数据权限
