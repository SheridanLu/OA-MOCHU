## 📝 核心业务规则

### 项目编号规则
- **实体项目**: P + 年份 + 6位序号（10位）
- **虚拟项目**: V + 年份 + 4位序号（8位）

### 合同编号规则
- **合同编号**: HT + 年月 + 4位序号（8位）

### 审批流程
1. **项目立项**: 采购员 → 财务 → 总经理
2. **收入合同**: 采购员 → 财务 → 法务 → 总经理
3. **支出合同**: 采购员 → 财务 → 法务 → 总经理
4. **劳务提报**: 项目经理 → 预算员 → 财务 → 总经理
5. **变更签证**: 项目经理 → 预算员 → 采购员 → 总经理

### 进度监控规则
- **产值占比 < 施工进度**: 黄色预警
- **申请产值 > 施工进度**: 红色锁定（无法提交）

### 竣工管理前置条件
- **竣工图纸审核**: 设计进度监控中遗留问题清零
- **竣工结算文件审核**: 工程对账单全部生成
